# 🤖 Resume Screening & Candidate Ranking System

## 📌 Project Overview

This project Resume Screening & Candidate Ranking System developed as part of the Future Interns Machine Learning Internship.

The system automatically compares resumes with a job description using Natural Language Processing (NLP), calculates a similarity score, ranks candidates, extracts skills, and identifies missing skills.

---

## 🎯 Objectives

- Automate resume screening
- Compare resumes with job descriptions
- Rank candidates based on relevance
- Extract technical skills
- Identify missing skills

---

## 🚀 Features

- Resume Text Cleaning
- NLP Preprocessing
- TF-IDF Vectorization
- Cosine Similarity Scoring
- Candidate Ranking
- Skill Extraction
- Missing Skill Identification
- Top Candidate Recommendation
- Data Visualization

---

## 🛠️ Technologies Used

- Python
- Pandas
- NLTK
- Scikit-learn
- Matplotlib
- Jupyter Notebook

---

## 📊 Dataset

- 2,484 Resumes
- 24 Job Categories

---

## 📈 Workflow

1. Load Resume Dataset
2. Clean Resume Text
3. Parse Job Description
4. Convert Text into TF-IDF Features
5. Compute Cosine Similarity
6. Rank Candidates
7. Extract Skills
8. Identify Missing Skills
9. Recommend Best Candidate
